# Mesa-updater
A simple bash script to check which is the newest mesa release, compare it with the installed one and automatically download, compile, and install it.

## How to use it
Be sure to have all the [compilation requirements from the installation tutorial](https://docs.mesa3d.org/install.html).

A sample bash script to install the requirements for debian-based systems is provided, but make sure to check out the official documentation.

___

Then simply run
```
sh get-mesa.sh
```
